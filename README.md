# Valentine's Day Interactive Page (My version)

A fun interactive Valentine’s page where the **"Yes" button grows every time "No" is clicked**, with animations, GIFs, and music.

This project is a **customized version** of the original by Sahil Gogna:  
https://github.com/SahilGogna/v-day

---

## About

This repository is a modified version of the original project, with changes such as:

- Visual and style adjustments  
- Custom text and messages  
- Tweaked interactions and behavior  

---

## Customization

- `index.html` – main page content  
- `yes.html` – success page  
- `script.js` – interactions 
- `style.css` – design and animations  
- `music/` – background audio
